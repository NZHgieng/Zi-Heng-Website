Final Project

Stage 3

Challenges

After identifying what I am going to create, hologram hourglass. I started finding the possible challenges of making it come to live.

List Of Challenges

  • Led not bright enough
  • Coding the led array
  • The led does not look good
  • 7 segment is small
  • There are not enough pins
  • Ultrasonic sensor to detect upward and downward motion
  • Hologram is block by transparent sand
  • Sand flow too fast or too slow
  • Sand stuck in the process of flowing

    • New List Of Challenges

    The following are the challenges that I discovered during the further stages.

  • Transfer signal to led array that is spinning
  • Servo motor not strong enough to turn
  • No enough pins
  • How to attached the servo motor to the hourglass

    • List Of Challenges - Explaination & Solution

  • Led not bright enough causing the hologram timing not able to display clearly.
  • Coding the led array is difficult.
  • The led does not look good so the solution is to change to using 7 segments.
  • The 7 segment is small so the solution is to change to using the led matrix.
  • There are not enough pins so the solution is to change back to using 7 segments.
  • I have a draft concept of how it is going to work, how to code but requires further testing to confirm.
  • It is unknown whether it will be too fast or too slow, requiring further testing to confirm.
  • It is unknown whether it will be bloak or not, and requires further testing to confirm.
  • Sand stuck in the process of flowing

    • New List Of Challenges - Explaination & Solution

  • The initial idea of the led array on the spinning hourglass is not practical so the solution is to change to placing it below the hourglass, on the overall outer container.
  • It is unknown, requires further testing to confirm as the servo motor needs to turn the hourglass which weighs about 300g. If it does not work, the solution is to change to using the dc motor. However, the code of the dc motor is unknown and how to attach it to the hourglass.
  • There are not enough pins for me to implement the led array on both sides of the hourglass so the solution is to change to only have it on one side of the hourglass.
  • I found the solution through watching the video below of how others use servo motors. The solution is to use screws to screw onto an acrylic board.
