Stage 4

Testing

Next, I started experimenting how to get about the challenges through various testing. Overall, most of it works well as initially plan but some needed to go with the alternative solution.

Testing List

  • Testing if the servo motor and relevant code are working.
  • Testing if the ultrasonic sensor and relevant code are working.
  • Testing if the plastic really can create holograms.
  • Testing if the acrylic can create holograms.
  • Testing if the sand will block the holograms.
  • Testing if the led can be displayed as holograms.
  • Testing if the 7 segments can be displayed as holograms.
  • Testing if the led matrix can be displayed as holograms.
  • Testing if the sand will be stuck.
  • Testing if the servo motor has enough power.
  • Testing if the hand signss to control the turning direction of the hourglass could be achieved.
  • Testing the size of the hole needs for the sand to flow for a minute.

    • Testing List- Results

  • The servo motor and its basic codes are working.
  • The ultrasonic sensor and its basic codes are working.
  • Plastic really can create holograms.
  • Acrylic can create holograms.
  • The sand will not block the holograms if the sand is not on the side of the display screen.
  • The led can be displayed as holograms.
  • The 7 segments can be displayed as holograms.
  • Unknown if the led matrix can be displayed as holograms because of not enough pins but based on the 7 segments testing, it is likely to work.
  • The sand will not be stuck if it did not touch any water related, having too small holes or objects stuck at the hole.
  • The servo motor has enough power to turn the entire set up.
  • The hand signs can control the turning direction of the hourglass, limited to up and down hand signs in respect to clockwise and anticlockwise direction.
  • The size of the hole needed for the sand to flow for a minute is 3mm diameter, being the most suitable to not get stuck easily.
